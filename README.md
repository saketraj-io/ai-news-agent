# AI News Agent

An AI-powered news aggregation and summarization application built with Streamlit. The application collects news from multiple sources, generates concise AI summaries, categorizes articles, and displays them through an interactive dashboard.

## Features

- 📰 Latest news collection
- 🤖 AI-powered news summaries
- 📂 News categorization
- 🔍 Search and filtering
- 📊 Interactive Streamlit dashboard
- ⚡ Simple and user-friendly interface

## Tech Stack

- Python
- Streamlit
- OpenAI / Gemini API
- Pandas
- News APIs

## Installation

Clone the repository:

```bash
git clone https://github.com/saketraj-io/ai-news-agent.git
cd ai-news-agent
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
